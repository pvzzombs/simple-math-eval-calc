# simple-math-eval-calc
A simple small code replacement for js eval

###What It Can Do ?
It can accept mathematical expression based on a string only.

###What Operations It Accept?
It accepts currently only:
<ul>
  <li> Addition </li>
  <li> Subtraction </li>
  <li> Multiplication </li>
  <li> Division </li>
  <li> Exponentation </li>
</ul>

###Examples of use:
```javascript
Eval(" 1 + 1"); //returns 2
Eval("10 - 5"); //returns 5
Eval("6 * 2"); //returns 12
Eval("14 / 7"); //returns 2
Eval("2 ^ 2"); //returns 4
```
